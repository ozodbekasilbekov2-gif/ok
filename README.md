# Xabar × Paynet — инвестиционная презентация (v3.3)

## Обзор проекта
- **Название**: Xabar — «телефон для того, у кого его нет»
- **Суть**: сервис в терминалах Paynet (Узбекистан): купюра 2 000 сум → готовое SMS получателю; всё выше 2 000 — обычное пополнение баланса
- **Цель сайта**: презентация для инвестора, раунд 360 млн сум (≈ $28K) за 12% (30 млн сум за 1%)

5. Dark insight — «таксофон убила стоимость инфраструктуры»
6. #product — 5 шагов + 3 сцены использования

## Встроенная мини-игра «Xabar Quest» (`/#xabar-quest`)
- Фотореалистичная визуальная новелла встроена прямо в инвестиционную презентацию — отдельной страницы игры больше нет
- 7 предоставленных пользователем фотосцен: дом, прохожий, магазин, владелец, меню Paynet, форма Xabar и успешный финал
- Нижняя часть каждого кадра перекрывается единым матовым диалоговым слоем, который скрывает исходные подписи и кнопки на фото
- Собственный анимированный текст, рабочие варианты выбора, индикатор 6 кадров, мягкие WebAudio-эффекты, перезапуск и полноэкранный режим
- Ветвящийся сценарий с основным и альтернативным финалами; на мобильных окно перестраивается в вертикальный формат

## 3D-сцены (Three.js 0.160)
- Тени: PCFSoftShadowMap, направленный свет 2048², cast/receive на всех вокселях
- Эпоха I: детализированная будка (стеклянные стены, приоткрытая дверь, вывеска, телефон с качающейся трубкой), очередь из 3 человек, скамейка, деревья, фонарь, облака
- Эпоха II: заброшенная накренившаяся будка, решётчатая вышка 6 ярусов с антеннами, пульсирующий маяк, расходящиеся кольца сигнала, человек с мобильным, вращающаяся «уходящая» монета
- Эпоха III: терминал Paynet (пульсирующий экран, клавиатура, анимация вставки купюры), отправитель + получатель вдали, летящий SMS-конверт по дуге, магазин с вывеской
- Idle-анимации людей (bobbing), дрейф облаков, drag-вращение + автоспин, GSAP elastic-переходы между эпохами

## Деплой

### Vercel (статика, рекомендуется для презентации)
- В репозитории есть `vercel.json` с `outputDirectory: "public"`
- В `public/index.html` лежит та же презентация, что и в `src/presentation.html`
- Импортируйте репозиторий в Vercel → Framework Preset: **Other** → Deploy
- Никакой сборки не требуется: Vercel раздаёт папку `public/` как статику
- Все ассеты (`/static/*`) кешируются на 1 год (`immutable`)

### Cloudflare Pages (оригинальная конфигурация)
```bash
npm run build
npx wrangler pages deploy
```
