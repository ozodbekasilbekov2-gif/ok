# Xabar — инвестиционная презентация

## Обзор проекта
- **Название**: Xabar («весть» по-узбекски)
- **Идея**: сервис в терминалах Paynet (Узбекистан) — человек без телефона вставляет купюру 2 000 сум и отправляет SMS; всё сверх 2 000 пополняет баланс получателя
- **Формат**: одностраничная инвестиционная презентация в стиле Claude AI (кремовый фон, serif-типографика, спокойная палитра)

## Секции
1. **Hero** — заголовок и CTA
2. **#history** — Minecraft-воксельная Three.js сцена эволюции связи: советская телефонная будка → 2003 (будка зарабатывает рядом с сотовой вышкой, Solan инвестирует $3 млн в «Ташкент Таксофони») → терминал Paynet сегодня. 3 вкладки-эпохи, вращение мышью
3. **#research** — deep research по доходности таксофонов: график Chart.js (парк США 2,6 млн → 100 тыс.; выручка $3,5 млрд 1989 → $286 млн 2015, FCC), факты по России/Великобритании/Японии, таймлайн
4. **#insight** — тёмная секция с выводом: спрос не умер, умерла себестоимость будки; Paynet = инфраструктура с нулевым CAPEX
5. **#product** — 6 шагов работы Xabar
6. **#economics** — разбор маржи (≈1 200 сум с SMS) + интерактивный калькулятор (объём/цена/комиссия Paynet)
7. **#ask** — раунд 650 млн сум, 2 транша, go/no-go критерии (конверсия ≥2%, доставка ≥95%, жалобы <0,2%)
8. **#finale** — тёмное закрытие: «Будка умерла. Смартфон её не заменил. Терминал Paynet — может»
9. **#sources** — все источники исследования (FCC, LA Times, Forbes, CNews, ComNews, Guardian, Profit.kz, Мининфоком, Paynet, ЦБ РУз, Eskiz)

## Юнит-экономика (базовый сценарий)
- Цена 2 000 сум → маржа ≈ 1 200 сум (60%): −НДС 12% −комиссия Paynet 440 −Eskiz 95 −OPEX 50
- Постоянные: 60 млн сум/мес; точка безубыточности ~1 930 SMS/день (1 SMS на 15 терминалов)
- Раунд: 650 млн сум ≈ $50k; окупаемость 20 / 7 / 2,6 мес по сценариям
