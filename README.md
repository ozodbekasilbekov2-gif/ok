# Xabar × Paynet — инвестиционная презентация (v3, editorial redesign)

## Обзор проекта
- **Название**: Xabar — «телефон для того, у кого его нет»
- **Суть**: сервис в терминалах Paynet (Узбекистан): купюра 2 000 сум → готовое SMS получателю; всё выше 2 000 — обычное пополнение баланса
- **Цель сайта**: презентация для инвестора, раунд 650 млн сум (≈ $51K)

## URLs
- **Sandbox (dev)**: https://3000-izopw40wcajltndxn5wii-3c7ff1b5.sandbox.novita.ai
- **Production**: не задеплоено (по запросу пользователя)

## Дизайн-система v3 (editorial)
- Палитра: cream #FAF9F5 / ink #1F1E1B / clay #C2562F / olive #6E7D4E / gold #B98A2E
- Шрифты: Source Serif 4 (заголовки/prose), Inter (текст), JetBrains Mono (цифры/подписи)
- 20+ видов элементов: sec-no (контурные номера секций), drop cap, marginalia, labeled rules, ledger-таблицы, checklist, rail-roadmap, fnote, era-tabs, 5 видов карточек, теговая система, бегунок прогресса скролла

## Структура (10 секций)
1. Hero — «Это не пополнение баланса. Это телефон…» + strip 28 000 / 96 трлн / 1 141
2. #story — история Мухтара (drop-cap prose + marginalia + steps-таймлайн + цитата)
3. #history — воксельная 3D-сцена (3 эпохи: таксофон → вышка → терминал Paynet) с era-tabs
4. #research — Chart.js график упадка таксофонов США 1989–2015 + 4 факта (Solan, Ростелеком, Ofcom, Япония) + цитата FCC
5. Dark insight — «таксофон убила стоимость инфраструктуры»
6. #product — 5 шагов + 3 сцены использования
7. #paynet — финансы партнёра (96 трлн, +55%, 395 млрд, 28 000) + аргумент 85–96 млн/мес
8. #economics — ledger-таблица маржи (2 000 → 1 141, 57%) + психология цены + интерактивный калькулятор
9. Homework — регуляторика / антифрод / SMS-каналы
10. #plan — rail-роадмап 14 недель + go/no-go · #ask — 650 млн, транши 350+300, 3 сценария окупаемости (17 / 6–7 / 2–3 мес) · финал · источники

## 3D-сцены (Three.js 0.160)
- Тени: PCFSoftShadowMap, направленный свет 2048², cast/receive на всех вокселях
- Эпоха I: детализированная будка (стеклянные стены, приоткрытая дверь, вывеска, телефон с качающейся трубкой), очередь из 3 человек, скамейка, деревья, фонарь, облака
- Эпоха II: заброшенная накренившаяся будка, решётчатая вышка 6 ярусов с антеннами, пульсирующий маяк, расходящиеся кольца сигнала, человек с мобильным, вращающаяся «уходящая» монета
- Эпоха III: терминал Paynet (пульсирующий экран, клавиатура, анимация вставки купюры), отправитель + получатель вдали, летящий SMS-конверт по дуге, магазин с вывеской
- Idle-анимации людей (bobbing), дрейф облаков, drag-вращение + автоспин, GSAP elastic-переходы между эпохами

## Экономика (данные из PPTX v4.0 пользователя)
